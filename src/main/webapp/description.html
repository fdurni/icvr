Service Description

The iCVR web service provides RESTful API. Use your browser's address bar to enter unformatted text that requires translation. Make sure that the text is no longer than 2000 characters.

Receive response from the service in three possible formats:

To get Plain Text formatted response use the following url: 

/service/txt/your_string

example: /service/txt/cvr

To get HTML formatted response use the following url: 

/service/html/your_string

example: /service/html/cvr

To get JSON formatted response use the following url: 

/service/json/your_string

example: /service/json/cvr
